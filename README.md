# streamlit
my streamlit place

## Environment.

- Python version "Python 3.9.4".
- Install packages:
  - Refer from requirement.txt
    - You can install packages by "pip install -r requirement.txt"

## How to create env?

1. create virtual environment.  
```
> python -m venv venv
```
2. activate
```
> .\venv\Script\activate
```
3. Install packages.
```
> pip install -r requirements.txt
```
